Foundations

Turn any crisis into clarity

The method is simple and powerful: Name it → Understand it → Meet the need. Apply it to anxiety, grief, conflict, burnout, shame, numbness, confusion, and despair.

Core Steps

The Framework

1) Name It

Labeling an emotion reduces threat quickly. Use gentle language: “This feels like fear,” “I’m carrying grief.”

2) Understand It

Ask: “What is this protecting? Which meaning or memory is touched?” Your nervous system always has a reason.

3) Meet the Need

Every state has a need—safety, warmth, belonging, rest, truth, movement. When honored, the state evolves.

Use This Script (anytime)

1) “Name: this feels like ___.”
2) “Why: my body is protecting me because ___.”
3) “Need: right now I need ___.”
4) “Act: the smallest next step I can take is ___.”
